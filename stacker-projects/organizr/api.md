# Organizr API Endpoints

## REST API
- Base URL: `http://localhost:9983/api/v2`
- Auth: API key (Settings → Main → API Key)

## Key Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | /api/v2/user | Current user info |
| GET | /api/v2/tabs | List tabs |
| POST | /api/v2/tabs | Create tab |
| GET | /api/v2/users | List users |
| GET | /api/v2/settings | Get settings |
