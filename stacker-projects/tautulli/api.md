# Tautulli API Endpoints

## REST API
- Base URL: `http://localhost:8181/api/v2`
- Auth: API key (Settings → Web Interface → API Key)

## Key Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | /api/v2?cmd=get_activity | Current activity |
| GET | /api/v2?cmd=get_history | Watch history |
| GET | /api/v2?cmd=get_libraries | Library stats |
| GET | /api/v2?cmd=get_users | User list |
| GET | /api/v2?cmd=get_servers | Server info |
| GET | /api/v2?cmd=get_settings | Settings |
